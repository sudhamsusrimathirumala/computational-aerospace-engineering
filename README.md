# Computational Aerospace Engineering and Simulation

## Overview

This project investigates how computational methods can be used to model aerospace systems.

The project focuses on developing a modular simulation framework capable of studying projectile motion, atmospheric flight, rocket ascent, and orbital mechanics.

## Research Questions

1. How can physical systems be simulated computationally?

2. How do numerical integration methods affect simulation accuracy?

3. How can aerodynamic and gravitational forces be modeled in software?

4. How can computational methods be applied to aerospace engineering problems?
